# 🍕 Pizza Sales Data Analysis (SQL Project)

---

### 🚀 Project Overview

This project analyzes a **pizza sales dataset** using SQL to extract meaningful business insights.

It helps in understanding:

* 📈 Sales performance
* 🧑‍🤝‍🧑 Customer ordering patterns
* 💰 Revenue contribution
* 🍕 Product popularity

---

### 📂 Project Structure

```bash
📦 Pizza-Sales-Analysis
 ┣ 📄 orders.csv
 ┣ 📄 order_details.csv
 ┣ 📄 pizzas.csv
 ┣ 📄 pizza_types.csv
 ┣ 🧾 Basic_Level_Questions.sql
 ┣ 🧾 Intermediate_Level_Questions.sql
 ┣ 🧾 Advances_Level_Questions.sql
 ┗ 📄 README.md
```

---

### 🛠️ Tools & Technologies Used

* **SQL (MySQL/PostgreSQL)** – Data analysis & querying
* **Excel/CSV** – Dataset storage

---

### 🔍 Key Analysis & Queries

The SQL analysis is divided into three levels:

#### 🟢 Basic Level

* Retrieve total number of orders
* Calculate total revenue
* Identify most common pizza size
* Find top ordered pizza types

---

#### 🟡 Intermediate Level

* Total quantity of each pizza category ordered
* Distribution of orders by hour
* Category-wise pizza distribution
* Average pizzas ordered per day
* Top 3 pizzas by revenue

---

#### 🔴 Advanced Level

* Percentage contribution of each pizza category to total revenue
* Cumulative revenue over time
* Top 3 pizzas per category using window functions

---

### 📊 Key Insights

#### 🍕 Product Performance

* Certain pizza categories contribute the highest revenue

#### ⏰ Customer Behavior

* Peak order hours help identify business rush timings

#### 📈 Sales Trends

* Revenue trends help in forecasting future sales

#### 💰 Business Impact

* Top-performing pizzas drive major revenue
